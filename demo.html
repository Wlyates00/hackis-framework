Hackis Documentation

This is a simple HTML page to demonstrate my simple Hackis framework! Right now only the most basic of components along with untilities are completed/started. The goal here is to have a tailwind inspired frameworks that provides preset components and styling features to allow for the development of easy and quick documentation pages.

  • Simple HTML structure: Legible HTML files
  • Basic CSS styling: Clean preset components with small flexibilities with utilities
  • Responsive design: Responsivity should be easily kept
